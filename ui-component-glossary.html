Glossário visual Hurby

Componentes de interface usados em SaaS e marketplace

Este arquivo serve para você aprender nomes técnicos e direcionar melhor o Codex, o designer ou o executor. Exemplo: em vez de dizer “box azul”, você pode dizer “card”, “panel”, “callout”, “hero”, “toolbar” ou “section”.

1. Section

Nome técnico: <section>
Bloco estrutural

Uma section é uma área do corpo da página. Pode conter cards, textos, formulários, tabelas ou qualquer conjunto de elementos.

Exemplo de section

Usada para separar uma parte da página, como “Pipeline Pro”, “Lista de imóveis” ou “Diagnóstico”.

2. Container

Nome técnico: container
Limite de largura

Container é a estrutura que segura o conteúdo centralizado com largura máxima. Normalmente envolve a página inteira ou grandes blocos.

Exemplo de container

Usado para não deixar o conteúdo colado nas bordas nem largo demais.
